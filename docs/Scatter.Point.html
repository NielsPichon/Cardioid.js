Class: Point

Scatter.Point(position)

Class describing a Point object. A point object is a p5.Vector with a color, a radius, a transform and a potential owner.

Constructor

new Point(position)

Parameters:
Name Type Description
position p5.Vector Position of the point.
Properties:
Name Type Attributes Default Description
rotation number 0 Rotation of the point. Useful when instancing a shape with this point's transform.
color Array.<number> 255,255,255,255 Color of the point in RGBA, defined as an array of 4 values in the [0, 255] range.
radius number 1 Radius of the point. Used as strokeWeight when drawing, as well as for relaxation.
scale p5.Vector 1,1 Point scale, useful when instancing a shape with this point's transform.
owner Scatter.Shape <optional>
 Shape to which this point belongs to.

Methods

copy() → {Scatter.Point}

Copies this point. This behaves as a deepcopy so the new point can be messed with without fear of backpropagating changes to this one. To avoid cyclic references due to the point possible ownership by a shape, we stayed away from using JSON Serialization-Deserialization an manually copied over properties instead.

falloff(vtx, decayPower) → {number}

Computes a falloff value based on the distance to the point. The falloff will reach zero when reaching the radius of the particle.
Parameters:
Name Type Description
vtx p5.Vector Point to evaluate the falloff at
