Class: Arc

Fresco.Arc(angle, radius, resolution, close)

Class describing an Arc from a collection of splines. By default the arc faces downwards and is centered on the vertical center line

Constructor

new Arc(angle, radius, resolution, close)

Parameters:
Name Type Default Description
angle number Arc angle in radians
radius number 50 Radius of the arc in pixels
resolution number 12 Number of points along the arc
close number false Whether to close the arc or not, linking the last point to the first
