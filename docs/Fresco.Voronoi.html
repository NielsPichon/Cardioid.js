Class: Voronoi

Fresco.Voronoi(numCells, gridBasedopt, is3Dopt)

A class for 2D/3D cellular noise

Constructor

new Voronoi(numCells, gridBasedopt, is3Dopt)

Parameters:
Name Type Attributes Default Description
numCells number Number of cells to create within the canvas. If `gridBased` is true, this is interpreted as the number of cell in each axis. (including the z axis if `is3D` is also true). Otherwise this is the number of points to scatter to generate cells
gridBased boolean <optional>
 false Whether to use a grid based approach. This will prevent animating individual cell centers but is much faster to compute.
is3D boolean <optional>
 false Whether to generate 3D noise. This will allow animating the z coordinate but is much slower.
Properties:
Name Type Description
gridBased number Whether to use a grid based approach.
numCells number Number of cells to create within the canvas.
is3D boolean Whether the noise is 3D noise.
seeds Array.<Fresco.Point> Seed points used to generate the noise
